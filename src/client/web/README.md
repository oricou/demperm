# Web Client

## Starting 

To lauch the web client, please use the following :

```bash
npm install
npm run dev
```
The app will be running at http://localhost:5173/

## Documentation

To generate a new documentation using jsdoc after updating the code, please run :

```bash
npm run createdoc

```

The documentation will be found at [./docs/client/web/docs](../../../docs/client/web/docs/index.html)
